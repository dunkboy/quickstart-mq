概要
ActiveMQ Apollo是一种更快，更可靠，更易于维护的消息传递代理，它是基于原始ActiveMQ的基础构建的。它使用截然不同的线程和消息调度体系结构来完成此任务 。

以目前的形式，Apollo仅支持STOMP协议，但就像原始的ActiveMQ一样，它被设计为多协议代理。在将来的版本中，它应该获得OpenWire支持，以便可以与ActiveMQ 5.x JMS客户端兼容。
